# Saiteja-Portfolio

A modern, responsive portfolio website showcasing skills, projects, and contact information.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations and transitions
- **Interactive Navigation**: Smooth scrolling navigation with active link highlighting
- **Mobile Menu**: Touch-friendly hamburger menu for mobile devices
- **Sections**:
  - Hero/Home section with introduction
  - About Me section with personal background
  - Skills & Technologies showcase
  - Featured Projects gallery
  - Contact information with multiple channels

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (ES6+)
- Modern web standards and best practices

## Getting Started

### Viewing the Website

Simply open `index.html` in a web browser to view the portfolio website.

### Local Development Server

To run a local development server:

```bash
# Using Python 3
python3 -m http.server 8080

# Then open http://localhost:8080 in your browser
```

## Customization

To customize this portfolio for your own use:

1. **Update Personal Information**: Edit `index.html` to update name, bio, and contact details
2. **Modify Skills**: Update the skills section with your own technologies and expertise
3. **Add Projects**: Replace the sample projects with your actual work
4. **Change Colors**: Edit CSS variables in `styles.css` (`:root` section) to match your brand
5. **Update Links**: Replace social media and contact links with your own
