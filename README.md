- **About**: Brief introduction and background

- **Projects**: Showcase of featured projects
- **Experience**: Timeline of work experience
- **Contact**: Contact information and message form

## How to Use

1. Clone the repository
2. Open `index.html` in a web browser
3. Customize the content with your own information:
   - Update personal information in `index.html`
   - Modify colors and styles in `styles.css`
   - Add your own projects, skills, and experience
   - Update social media links
   - Replace contact information with your own

## Customization

### Colors
The color scheme can be easily modified in `styles.css` by changing the CSS variables:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --dark-bg: #0f172a;
    --light-bg: #1e293b;
    --text-primary: #f1f5f9;
    --text-secondary: #cbd5e1;
}
```

### Content
- Update your name, title, and description in the hero section
- Add your actual projects with links and descriptions
- Update the skills section with your technologies
- Modify the experience timeline with your work history
- Change contact information to your own
